## 🚀 Getting Started

No frameworks or build tools required. Just a browser.
```bash
# Clone the repository
git clone https://github.com/your-username/berger-hut.git

# Navigate into the project folder
cd berger-hut

# Open in browser
open index.html
```

Or simply double-click `index.html` to open it in your browser.

---

## 🛠️ Built With

- **HTML5** – Page structure
- **CSS3** – Styling and responsive layout
- No frameworks, no JavaScript

---

## ✨ Features

- Dark themed design throughout
- Hero section with full-screen background image
- About Us section
- Menu section with Appetizers, Main Courses, and Desserts
- Image gallery with CSS grid layout
- Contact section with address, phone, email and a contact form
- Responsive footer with Privacy Policy and Terms of Service links
